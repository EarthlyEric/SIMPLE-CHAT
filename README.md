# SIMPLE-CHAT[![Run on Repl.it](https://repl.it/badge/github/EarthlyEric/SIMPLE-CHAT)](https://repl.it/github/EarthlyEric/SIMPLE-CHAT)
# 功能
1.登入頁面
2.依賴程式庫已打包在程式碼發布包中
3.登入和登出自動顯示在公共頻到道中
4.自動註記訊息發出時間
5.高度可自定義
